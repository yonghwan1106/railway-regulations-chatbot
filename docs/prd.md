# AI 기반 철도건설규정 통합검색 챗봇 PRD

## 제품 개요
AI 챗봇이 철도건설 설계·검토 실무에서 필요한 규정·근거를 사용자 질의에 맞춰 빠르게 찾아주고, 상충 규정 점검을 지원하는 B2B/B2G 전문 서비스입니다.

## 주요 기능
- 자연어 규정 질의응답 챗봇
- 상충 규정 자동 검토 및 안내
- 규정 데이터 통합 검색(챗봇 기반)
- (옵션) 즐겨찾기, 변경 알림 기능

## 타겟 사용자
- 철도건설 설계사·엔지니어
- 공공기관 담당자
- 중소 설계업체 실무자

## 주요 사용 사례
- 설계 중 규정 빠른 확인 및 근거 명시 정보 얻기
- 상충 규정 자동 감지/경고로 설계 검증 간소화

## Pain Point & 사용자 목표
- 규정 해석 및 상충 내용 파악의 어려움 → AI가 답변과 관련 근거/문서/경고 자동 제시
- 복잡한 자료 여러 개를 직접 찾는 시간 낭비 → 1회 즉시질문/답변 구조로 해결
- **핵심 목표:** 규정 정보 확인에 드는 시간·노력 최소화

## 비슷한 서비스
- 로톡, 리걸브레인 등 AI 법률 질의/답변 플랫폼

## KPI
- 답변 정확도 90% 이상
- 평균 응답시간 3초 이내
- 반복 사용률 60% 이상

## 개발 전략
- MVP로 챗봇 Q&A/상충 검토, 가벼운 DB 구축부터 우선 구현 후 점진 고도화
- Web(PC/모바일) 반응형 UI, 기본 음성 질의 지원 검토

## 참고
- 베이스 기술스택: Next.js 15, shadcn, lucide-react, typescript, tailwindcss, supabase, @tanstack/react-query 등 EasyNext 템플릿 사용
