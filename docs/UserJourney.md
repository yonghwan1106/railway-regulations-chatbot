# AI 철도규정챗봇 사용자 여정 (User Journey)

## 1. 주요 사용자 페르소나
- 설계엔지니어(김철수, 32세)
- 공단 담당자(박영희, 28세)
- 중소 설계업체 대표/실무자(이스타트, 35세)

## 2. 대표 사용자 플로우(시나리오)

### 1단계: 로그인/접속
- 웹(PC/모바일) 접속 → 기본 안내 확인

### 2단계: 규정 검색/질문 입력
- 자연어로 원하는 규정/상황 질문 입력 (예 “250km/h 곡선반지름?”)
- (옵션) 마이크로 음성 입력
- 질문 템플릿/추천 예시도 활용

### 3단계: 규정 답변 및 상충 점검
- 챗봇이 관련 규정/근거/수치 즉시 응답
- 상충 내용 있을 시 경고 및 우선권 조항/설명 안내

### 4단계: 추가질의/문맥 연계
- 사용자는 추가 질문, 맥락 이어받아 자연스럽게 대화

### 5단계: 정보 저장/활용
- ‘즐겨찾기’, ‘내 검색기록’ 저장 (옵션)
- 업무 보고, 검토자료로 직접 복사/활용

### 6단계: 규정 변경 알림 (옵션)
- 특정 규정/분야 구독 시, 개정 알림 수신

## 3. 기대 사용자 행동
- 직접 문서 찾는 시간 절감
- 반복검색/즐겨찾기 활용도 증가
- 규정해석 오류 감소, 업무신뢰도 향상
- 챗봇 대화/검색 결과 즉시 활용 가능
